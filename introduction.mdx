**Miso** is a small, production-ready, component-oriented, isomorphic [Haskell](https://www.haskell.org/) front-end framework for quickly building highly interactive single-page web and mobile applications.

It features a virtual DOM, recursive diffing/patching algorithm, attribute and property normalization, event delegation, event batching, SVG, 2D/3D Canvas (WebGL), Server-sent events (SSE), WebSockets, type-safe [servant](https://haskell-servant.github.io/)-style routing and an extensible subscription-based subsystem. Inspired by [Elm](http://elm-lang.org/) and [React](http://react.dev/).

## Why miso?

<CardGroup cols={2}>
  <Card title="Virtual DOM" icon="layer-group">
    High-performance recursive diffing and patching with 100% test coverage on the core engine
  </Card>
  <Card title="Pure by default" icon="shield-check">
    Side effects are explicit and typed via the `Effect` monad — no surprises
  </Card>
  <Card title="WASM & JS targets" icon="microchip">
    Compile to WebAssembly or JavaScript using standard GHC tooling
  </Card>
  <Card title="Type-safe routing" icon="route">
    Servant-style routing with correct-by-construction URL encoding and decoding
  </Card>
  <Card title="Component system" icon="puzzle-piece">
    Nest components with typed parent-child bindings, lifecycle hooks, and data bindings
  </Card>
  <Card title="Isomorphic" icon="arrows-rotate">
    Render on the server, hydrate on the client — one codebase for both
  </Card>
</CardGroup>

## Compilation targets

Miso supports three compilation backends:

| Backend | Compiler | Use case |
|---------|----------|----------|
| **WebAssembly** | `wasm32-wasi-ghc` | Recommended default — full GHC, best tooling |
| **JavaScript** | `javascript-unknown-ghcjs-ghc` | GHCJS-compatible JS output |
| **Vanilla GHC** | Standard `ghc` | Server-side rendering and testing |

## Real-world examples

Miso has been used in quantitative finance, network security, defense research, SaaS products, and academic settings. The largest known installation runs ~200,000 lines of miso code serving 10,000+ users.
