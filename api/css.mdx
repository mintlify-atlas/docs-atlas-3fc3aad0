`Miso.CSS` provides a typed DSL for building CSS stylesheets and inline styles. It integrates with the miso component model through `style_` and `styleInline_` attributes, and supports full stylesheet injection via `StyleSheet`.

## Core Types

```haskell
-- | A single CSS property-value pair
type Style = (MisoString, MisoString)

-- | A CSS rule set: selector + styles, keyframes, or media query
data Styles
  = Styles   (MisoString, [Style])
  | KeyFrame MisoString [(MisoString, [Style])]
  | Media    MisoString [(MisoString, [Style])]
  deriving (Eq, Show)

-- | A collection of CSS rules
newtype StyleSheet = StyleSheet { getStyleSheet :: [Styles] }
  deriving (Eq, Show)
```

## Smart Constructors

```haskell
-- | Build a Styles rule from a selector and property list
selector_ :: MisoString -> [Style] -> Styles

-- | Build a StyleSheet from a list of Styles rules
sheet_ :: [Styles] -> StyleSheet

-- | @keyframes rule
keyframes_ :: MisoString -> [(MisoString, [Style])] -> Styles

-- | @media rule
media_ :: MisoString -> [(MisoString, [Style])] -> Styles
```

## Attribute Helpers

```haskell
-- | Set the style attribute from a typed [Style] list
-- (replaces any styles not in the list)
style_ :: [Style] -> Attribute action

-- | Set the style attribute from a raw CSS string
styleInline_ :: MisoString -> Attribute action

-- | Render a StyleSheet to a MisoString (for <style> tags)
renderStyleSheet :: StyleSheet -> MisoString
```

## Miso.CSS.Color

The `Color` type is re-exported from `Miso.CSS`.

```haskell
data Color
  = RGBA Int Int Int Double  -- ^ rgba(r,g,b,a)
  | RGB  Int Int Int         -- ^ rgb(r,g,b)
  | HSL  Int Int Int         -- ^ hsl(h,s,l)
  | HSLA Int Int Int Double  -- ^ hsla(h,s,l,a)
  | Hex  MisoString          -- ^ #rrggbb
  | VarColor MisoString      -- ^ var(--name)

-- Smart constructors
rgba :: Int -> Int -> Int -> Double -> Color
rgb  :: Int -> Int -> Int -> Color
hsl  :: Int -> Int -> Int -> Color
hsla :: Int -> Int -> Int -> Double -> Color
hex  :: MisoString -> Color
var  :: MisoString -> Color  -- CSS variable color

-- Render to MisoString
renderColor :: Color -> MisoString
```

Over 140 named CSS colors are exported: `red`, `blue`, `green`, `white`, `black`, `transparent`, `aliceblue`, `cornflowerblue`, etc.

```haskell
-- OverloadedLabels syntax for hex colors
grey :: Color
grey = #cccccc
```

## Full Example

```haskell
{-# LANGUAGE OverloadedStrings #-}
module Main where

import Miso
import Miso.CSS
import Miso.String (ms)

-- Define a stylesheet
myStyles :: StyleSheet
myStyles = sheet_
  [ selector_ ".container"
      [ display "flex"
      , flexDirection "column"
      , alignItems "center"
      , padding (px 16)
      , backgroundColor (rgba 245 245 245 1)
      ]
  , selector_ ".title"
      [ fontSize (px 24)
      , fontWeight "bold"
      , color (hex "333333")
      , marginBottom (px 8)
      ]
  , selector_ ".button"
      [ backgroundColor blue
      , color white
      , padding (px 8 <> " " <> px 16)
      , borderRadius (px 4)
      , cursor "pointer"
      ]
  , keyframes_ "fade-in"
      [ "from" =: [ opacity "0" ]
      , "to"   =: [ opacity "1" ]
      ]
  , media_ "(max-width: 600px)"
      [ ".container" =: [ flexDirection "column" ]
      ]
  ]

-- Render the stylesheet into a <style> tag
viewModel :: Model -> View Model Action
viewModel model =
  div_ []
    [ -- Inject stylesheet
      node HTML "style" [] [ text (renderStyleSheet myStyles) ]
      -- Use typed inline styles
    , div_
        [ style_
            [ display "flex"
            , backgroundColor (rgba 0 128 255 0.1)
            , borderRadius (px 8)
            , padding (px 12)
            ]
        ]
        [ text "Styled content" ]
    ]
```
