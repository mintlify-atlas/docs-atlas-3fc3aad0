---
title: "Miso.Svg"
description: "SVG rendering support in miso"
---

`Miso.Svg` provides SVG element constructors and events. SVG elements are rendered in the `http://www.w3.org/2000/svg` namespace, so they can be embedded directly in your miso HTML views.

```haskell
import Miso
import Miso.Svg
```

## Quick Example

```haskell
import Miso
import Miso.Svg
import Miso.Svg.Element

myIcon :: View action
myIcon =
  svg_ [ height_ "100", width_ "100" ]
    [ circle_
        [ cx_ "50", cy_ "50", r_ "40"
        , stroke_ "green", strokeWidth_ "4", fill_ "yellow"
        ]
        []
    ]
```

## SVG Events

SVG elements support the same event attributes as HTML elements (`onClick_`, `onMouseEnter_`, etc.) through `Miso.Svg.Event`.

## Embedding in HTML

SVG elements can be placed anywhere in your HTML view tree:

```haskell
viewModel :: Model -> View Model Action
viewModel model =
  div_ [ class_ "container" ]
    [ h1_ [] [ "SVG Demo" ]
    , svg_ [ width_ "200", height_ "200", viewBox_ "0 0 200 200" ]
        [ -- Background
          rect_ [ x_ "0", y_ "0", width_ "200", height_ "200", fill_ "lightgray" ] []
          -- A triangle
        , polygon_
            [ points_ "100,20 180,180 20,180"
            , fill_ "steelblue"
            , stroke_ "navy"
            , strokeWidth_ "3"
            ]
            []
          -- Centered text
        , text_
            [ x_ "100", y_ "105"
            , textAnchor_ "middle"
            , fill_ "white"
            , fontSize_ "16"
            ]
            [ "Hello SVG" ]
        ]
    ]
```

<Note>
  All SVG element constructors produce nodes in the SVG namespace (`http://www.w3.org/2000/svg`). This is handled automatically — you do not need to set any namespace attribute yourself.
</Note>
