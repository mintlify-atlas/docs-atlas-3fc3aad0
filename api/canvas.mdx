`Miso.Canvas` provides a typed DSL for the [HTML5 Canvas 2D API](https://developer.mozilla.org/en-US/docs/Web/API/CanvasRenderingContext2D). Drawing operations run inside a `Canvas` monad backed by `ReaderT CanvasContext2D IO`.

## Core Types

```haskell
-- | The Canvas drawing monad
type Canvas a = ReaderT CanvasContext2D IO a

-- | Underlying JS context value
type CanvasContext2D = JSVal

-- | An (x, y) coordinate pair
type Coord = (Double, Double)

-- | Color, gradient, or pattern for fill/stroke styles
data StyleArg
  = ColorArg Color
  | GradientArg Gradient
  | PatternArg Pattern

newtype Pattern  = Pattern JSVal
newtype Gradient = Gradient JSVal
newtype ImageData = ImageData JSVal
```

### Smart Constructors for StyleArg

```haskell
color    :: Color    -> StyleArg
gradient :: Gradient -> StyleArg
pattern_ :: Pattern  -> StyleArg
```

### Enum Types

```haskell
data LineCapType  = LineCapButt | LineCapRound | LineCapSquare
data LineJoinType = LineJoinBevel | LineJoinRound | LineJoinMiter
data DirectionType = LTR | RTL | Inherit
data TextAlignType = TextAlignCenter | TextAlignEnd | TextAlignLeft
                   | TextAlignRight   | TextAlignStart
data TextBaselineType
  = TextBaselineAlphabetic | TextBaselineTop    | TextBaselineHanging
  | TextBaselineMiddle     | TextBaselineIdeographic | TextBaselineBottom
data CompositeOperation
  = SourceOver | SourceAtop | SourceIn  | SourceOut
  | DestinationOver | DestinationAtop | DestinationIn | DestinationOut
  | Lighter | Copy | Xor
data PatternType = Repeat | RepeatX | RepeatY | NoRepeat
```

## Canvas Elements

Two element constructors are provided:

```haskell
-- | Canvas element backed by the 2D context monad (ReaderT-based)
canvas
  :: (FromJSVal canvasState, ToJSVal canvasState)
  => [Attribute action]
  -> (DOMRef -> Canvas canvasState) -- ^ init: receive DOM ref, return initial state
  -> (canvasState -> Canvas ())      -- ^ draw: called each render
  -> View model action

-- | Canvas element for raw JSVal contexts (e.g. three.js, WebGL)
canvas_
  :: (FromJSVal canvasState, ToJSVal canvasState)
  => [Attribute action]
  -> (DOMRef -> IO canvasState)  -- ^ init
  -> (canvasState -> IO ())       -- ^ draw
  -> View model action
```

Use `canvas` for the standard 2D context. Use `canvas_` when you need raw `IO` access (e.g. for WebGL or third-party libraries).

## Full Example

```haskell
import Miso
import Miso.Canvas
import Miso.CSS.Color (red, blue)

type Model = ()
data Action = NoOp

viewModel :: Model -> View Action
viewModel _ =
  div_ []
    [ canvas
        [ width_ "400", height_ "300" ]
        -- Init: get context, return canvas size as state
        (\_ -> pure (400.0, 300.0))
        -- Draw: called each render
        (\(w, h) -> do
            -- Clear
            fillStyle (color red)
            fillRect (0, 0, w, h)

            -- Blue circle
            beginPath ()
            arc (w / 2, h / 2, 80, 0, 2 * pi)
            fillStyle (color blue)
            fill ()

            -- Text
            font "bold 24px sans-serif"
            fillStyle (color (Hex "ffffff"))
            textAlign TextAlignCenter
            fillText ("Hello Canvas!", w / 2, h / 2 + 8)
        )
    ]
```
